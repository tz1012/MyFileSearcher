Gemini File Search App
======================

설치 및 실행 방법
------------------

1. GeminiFileSearch.exe 파일을 실행하세요.
2. 검은색 콘솔 창이 열리고 잠시 후 브라우저가 자동으로 실행됩니다.
   (브라우저가 열리지 않으면 http://localhost:5000 주소로 접속하세요)
3. 프로그램 왼쪽 사이드바에 Gemini API Key를 입력하고 'Set'을 누르세요.
   (API Key는 https://aistudio.google.com/app/apikey 에서 무료로 발급받을 수 있습니다)
4. 사용하실 문서를 드래그하여 업로드한 뒤 대화를 시작하세요.

주의사항
--------
- 이 프로그램은 로컬 웹 서버를 구동합니다.
- 프로그램 종료 시 콘솔 창을 닫아주세요.
- 처음 실행 시 바이러스 백신이 경고할 수 있으나, Python으로 개발된 서명되지 않은 앱이라 발생하는 현상입니다.
